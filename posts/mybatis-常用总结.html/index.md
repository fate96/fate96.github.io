### 使用 like 进行模糊批评

当使用 sql 语句进行模糊查询时，由于模糊查询需要使用 % 符号表示模糊查询，所以在查询的过程中需要传入 % 字符。

两种使用方式

1. like "%"#{XX}"%" ：手动拼接，因为 #{} 作为占位符，会被单引号的字符串直接替换，所以 % 需要使用双引号的字符串
2. 使用concat()函数连接参数形式：like concat(''%', #{xx},'%')
